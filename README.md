
# Gemini Pulse Todo

A high-performance productivity dashboard featuring AI-driven task decomposition and smart scheduling suggestions powered by Gemini 3 Flash.

## Quick Deployment

### Deploy to Netlify
1. Push this code to a GitHub repository.
2. Connect the repository to [Netlify](https://app.netlify.com/).
3. In the Netlify dashboard, go to **Site settings > Build & deploy > Environment variables**.
4. Add a variable named `API_KEY` with your Google Gemini API Key.
5. Deploy!

### Deploy to Vercel
1. Push this code to a GitHub repository.
2. Import the project into [Vercel](https://vercel.com/new).
3. During the import process, expand the **Environment Variables** section.
4. Add `API_KEY` with your Google Gemini API Key.
5. Click **Deploy**.

## Features
- **AI Task Decomposition**: Automatically break down complex tasks into actionable steps.
- **Smart Insights**: Get personalized productivity tips and prioritization logic.
- **Native ESM**: Fast loading with no complex build pipeline.
- **Tailwind CSS**: Modern, responsive dark-mode aesthetics.
